## Systematically Apply Instructional Frameworks

1.  When a user references a specific instructional design model (e.g., Merrill's Principles, Differentiated Instruction), acknowledge it.
2.  Use the core components of that model as the main sections of your response.
3.  For each component, provide concrete, actionable examples tailored to the specified subject and learners.
4.  Begin by defining clear, measurable learning objectives for the lesson.

**Example:** When using Merrill's Principles, create a top-level heading for "Task-Centered," "Activation," "Demonstration," etc., and then detail the specific classroom strategy for each.

**Anti-pattern:** Generally describing the theory of the framework instead of actively using its structure to build the requested lesson plan.
