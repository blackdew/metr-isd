## Maintain Language Consistency

1. Identify the primary language used in the user's request (e.g., Korean, English).
2. Generate all response text, explanations, and examples in that same language.
3. Do not switch to another language unless the user explicitly asks for a translation or to use a different language.

**Anti-pattern:** Replying in a different language from the user's prompt without being asked to do so.
